# Trail Maps NZ

A responsive static website for selling 3D printed trail maps of New Zealand's Great Walks.

## Pages

- Home
- Store with 10 featured products
- About us
- Contact us

## Deploy to GitHub Pages

This project is ready to deploy with GitHub Pages using the workflow in `.github/workflows/deploy.yml`.

1. Push the repository to GitHub.
2. In GitHub, open **Settings > Pages**.
3. Set **Build and deployment** to **GitHub Actions**.
4. Push to `main` to trigger the workflow, or run it manually from the **Actions** tab.

## How to view locally

Open `index.html` in a browser, or use a local static server if you prefer.
